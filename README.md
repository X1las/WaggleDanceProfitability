# WaggleDanceProfitability
Repo containing code for the profitability model on the 'Waggle Dance' of european honeybees

Data sources are aquired from the original study by P. L. Kohl and B. Rutschmann from 2021 on "Honey bees communicate distance via non-linear waggle duration functions" as seen here: https://www.ncbi.nlm.nih.gov/pmc/articles/PMC8029670/
